How to Get an API Key

  1. Go to VirusTotal.com
  2. Sign up for a free account (or log in if you have one)
  3. Navigate to your profile settings
  4. Go to the "API Key" section
  5. Copy your API key and paste it above
Note: Free VirusTotal API keys have rate limits. You can make up to 4 requests per minute and 500 requests per day.
